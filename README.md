### Sudoku with Compose
An offline-first Sudoku game made with Jetpack compose. It uses Kudoku Sudoku engine in the backend.
Gametype supports:
- Classic 9x9 gameplay
- Mistakes detection and highlighting

#### Architecture:
Sudoku demonstrates modern Android development with Jetpack compose, Hilt, Material3, Coroutines, Flow, RoomDB, ViewModel and is based on MVVM architecture and 
Repository pattern.
